# Infinite Runner OpenGL

A game made entirely in OpenGL.

## Install 

Run: 
```bash
  sudo bash install.bash
```

and install libraries as needed. GLAD already comes with the program in the form of the glad.c file.

To build run 

```bash
   mkdir build
   cmake -B build
   cmake --build ./build
```

and run the game executable in the build folder.

```bash
    cd build
    ./game
```
